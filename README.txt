README for "dinamic_crescent_order" C++ application
Created by r4m0nl1m4 03/03/2020

No terminal, chame o makefile.

  Para compilação:	

    <user_name>@<host_name>$ make

  Para limpeza dos arquivos de compilação:

    <user_name>@<host_name>$ make clean

Para excução:

  ./dinamic_crescent_order <número_inteiro>

*runtime with Ubuntu 'time' command
